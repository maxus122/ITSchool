Отрисовка персонажа ,,Вжиг" - герой мультсериала «Чип и Дейл спешат на помощь».

Аргументы
xХ-координата точки
yY-координата точки
flyHandНаличие рук лапок у персонажа (0 - Нет рук, 1 - Правая рука, 2 - Левая рука, 3 - Обе руки)
flyLegНаличие ног у персонажа (0 - Нет ног, 1 - Правая нога, 2 - Левая нога, 3 - Обе ноги) (При отсутствии ног, крылья автоматические исчезают)
flyScaleРазмер персонажа (1 - Стандартный размер, 2 - В 2 раза больше стандартного размера и т.д.)
flyHairНаличие волос у персонажа (0 - Нет волос, 1 - Есть волос)
flyWingsНаличие крыльев у персонажа (0 - Нет крыльев, 1 - Есть крылья)
flyEyesНаличие глаз у персонажа (0 - Нет глаз, 1 - Есть глаза)
flyRothColorЦвет рта персонажа
flyShirtColorЦвет футболки персонажа
flySkinColorЦвет кожи персонажа
flyEyesColorЦвет глаз персонажа
Предупреждения
Если цвета разных частей тела совпадают, может произойти ядерный коллапс могут наблюдаться серъёзные графические баги.
Возвращает
Если операция была успешна - true, иначе - false.
Заметки
Для использования функции нужно создать холст из TXLib.

Пример использования функции смотрите в примере реализации.

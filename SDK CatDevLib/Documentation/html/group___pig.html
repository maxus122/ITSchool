Отрисовка персонажа ,,Пятачок" - герой мультфильма «Винни-Пух».

Аргументы
xХ-координата точки
yY-координата точки
pigEarНаличие ушей у персонажа (0 - Нет ушей, 1 - Правое ухо, 2 - Левое ухо, 3 - Оба уха)
pigHandНаличие рук лапок у персонажа (0 - Нет рук, 1 - Правая рука, 2 - Левая рука, 3 - Обе руки)
pigScaleРазмер персонажа (1 - Стандартный размер, 2 - В 2 раза больше стандартного размера и т.д.)
pigEyesНаличие глаз у персонажа (0 - Нет глаз, 1 - Есть глаза)
pigLegНаличие я в душе не знаю, что там у свиней: ноги, лапы, копыта ног у персонажа (0 - Нет ног, 1 - Правая нога, 2 - Левая нога, 3 - Обе ноги)
pigLeftArmНаличие левой руки (0 - Нет левой руки, 1 - Есть левая рука) (Изминение наличия правой руки отсутствует)
pigRothНаличие рта у персонажа (0 - Нет рта, 1 - Есть рот)
pigEarColorЦвет ушей персонажа
pigNoseColorЦвет носа пятачка персонажа
pigShirtColorЦвет футболки персонажа
pigSkinColorЦвет кожи персонажа
Предупреждения
Если цвета разных частей тела совпадают, может произойти ядерный коллапс могут наблюдаться серъёзные графические баги.
Возвращает
Если операция была успешна - true, иначе - false.
Заметки
Для использования функции нужно создать холст из TXLib.

Пример использования функции смотрите в примере реализации.

Отрисовка персонажа ,,Миньон" - герой мультфильма «Гадкий я».

Аргументы
xХ-координата точки
yY-координата точки
MinionHandНаличие рук у персонажа (0 - Нет рук, 1 - Правая рука, 2 - Левая рука, 3 - Обе руки)
MinionScaleРазмер персонажа (1 - Стандартный размер, 2 - В 2 раза больше стандартного размера и т.д.)
MinionGlassesНаличие очков у персонажа (0 - Нет очков, 1 - Есть очки)
MinionHairНаличие волос у персонажа (0 - Нет волос, 1 - Есть волос)
MinionRothНаличие рта у персонажа (0 - Нет рта, 1 - Есть рот)
MinionEyesColorЦвет глаз персонажа
MinionPantsColorЦвет штанов персонажа
MinionGlassesColorЦвет очков персонажа
MinionPocketColorЦвет карманчика на штанах персонажа
Предупреждения
Если цвета разных частей тела совпадают, может произойти ядерный коллапс могут наблюдаться серъёзные графические баги.
Возвращает
Если операция была успешна - true, иначе - false.
Заметки
Для использования функции нужно создать холст из TXLib.

Пример использования функции смотрите в примере реализации.

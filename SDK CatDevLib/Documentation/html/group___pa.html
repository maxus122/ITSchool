Отрисовка персонажа ,,Прапор" - герой мультфильма «Пингвины Мадагаскара».

Аргументы
xХ-координата точки
yY-координата точки
penguinHandНаличие рук у персонажа (0 - Нет рук, 1 - Правая рука, 2 - Левая рука, 3 - Обе руки)
penguinLegНаличие ног у персонажа (0 - Нет ног, 1 - Правая нога, 2 - Левая нога, 3 - Обе ноги)
penguinScaleРазмер персонажа (1 - Стандартный размер, 2 - В 2 раза больше стандартного размера и т.д.)
penguinRothНаличие рта у персонажа (0 - Нет рта, 1 - Есть рот)
penguinEyesНаличие глаз у персонажа (0 - Нет глаз, 1 - Есть глаза)
penguinEyesColorЦвет глаз у персонажа
penguinLegsColorЦвет ног у персонажа
penguinBackColorЦвет спины у персонажа
penguinBallyColorЦвет пузика у персонажа)))
Предупреждения
Если цвета разных частей тела совпадают, может произойти ядерный коллапс могут наблюдаться серъёзные графические баги.
Возвращает
Если операция была успешна - true, иначе - false.
Заметки
Для использования функции нужно создать холст из TXLib.

Пример использования функции смотрите в примере реализации.

Отрисовка персонажа ,,Тимон" - герой анимационного фильма «Король Лев».

Аргументы
xХ-координата точки
yY-координата точки
timonEarНаличие ушей у персонажа (0 - Нет ушей, 1 - Правое ухо, 2 - Левое ухо, 3 - Оба уха)
timonLegНаличие ног у персонажа (0 - Нет ног, 1 - Правая нога, 2 - Левая нога, 3 - Обе ноги)
timonScaleРазмер персонажа (1 - Стандартный размер, 2 - В 2 раза больше стандартного размера и т.д.)
timonTaleНаличие хвоста у персонажа (0 - Нет хвоста, 1 - Есть хвост)
timonHairНаличие волос у персонажа (0 - Нет волос, 1 - Есть волос)
timonRothНаличие рта у персонажа (0 - Нет рта, 1 - Есть рот)
timonBallyColorЦвет пузика персонажа)))
timonBackColorЦвет спины персонажа
timonHairColorЦвет волос персонажа
timonLimbColorЦвет конечностей персонажа
Предупреждения
Если цвета разных частей тела совпадают, может произойти ядерный коллапс могут наблюдаться серъёзные графические баги.
Возвращает
Если операция была успешна - true, иначе - false.
Заметки
Для использования функции нужно создать холст из TXLib.

Пример использования функции смотрите в примере реализации.

Отрисовка персонажа ,,Скуби-Ду" - герой мультсериала «Скуби-Ду шоу».

Аргументы
xХ-координата точки
yY-координата точки
SkubiScaleРазмер персонажа (1 - Стандартный размер, 2 - В 2 раза больше стандартного размера и т.д.)
SkubiKeychainНаличие брелка у персонажа (0 - Нет брелка, 1 - Есть брелок)
SkubyMolesНаличие родинок у персонажа (0 - Нет родинок, 1 - Есть родинки)
SkubiEyesНаличие глаз у персонажа (0 - Нет глаз, 1 - Есть глаза)
SkubiNoseНаличие носа у персонажа (0 - Нет носа, 1 - Есть нос)
SkubiTeethНаличие зубов у персонажа (0 - Нет зубов, 1 - Есть зубы)
SkubiStrokePanelColorЦвет обводки у персонажа
SkubiSkinColorЦвет кожи персонажа Внимание: Мы против рассизма!
SkubiCollarColorЦвет ошейника у персонажа
SkubiEyesColorЦвет глаз у персонажа
Предупреждения
Если цвета разных частей тела совпадают, может произойти ядерный коллапс могут наблюдаться серъёзные графические баги.
Возвращает
Если операция была успешна - true, иначе - false.
Заметки
Для использования функции нужно создать холст из TXLib.

Пример использования функции смотрите в примере реализации.

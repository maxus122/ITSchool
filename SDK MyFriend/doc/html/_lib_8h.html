[in]shoesFinesРазмер правого кроссовка Финеса.
[in]pantsFinesДлинна шорт Финеса.
[in]moodНастроение Финеса. True - улыбается, False - грустит.
[in]eyesFinesОткрыть/закрыть глаза Финеса. True - глаза открыты, False - глаза открыты.
[in]haireFinesРисовать/не рисовать волосы Финеса. True - волсы рисуются, False - волосы не рисуются.
[in]reverseColorПоменять местами цвет футболки и шорт. True - шорты синие, футболка оранжевая, False - шорты оранжевые, футболка синяя.
[in]legsFinesРисовать/не рисовать ноги Финеса. True - шорты, ноги, кроссовки рисуются, False - шорты, ноги, кроссовки не рисуются.
[in]bodyFinesРисовать/не рисовать туловище Финеса. True - голова, торс, руки, футболка рисуются, False - голова, торс, руки, футболка рисуются.
[in]scaleFinesМасштаб персонажа.
Предупреждения
Параметр shoesFines изменяет размер только правого кроссовка. P.S. сохраняйтесь чаще.

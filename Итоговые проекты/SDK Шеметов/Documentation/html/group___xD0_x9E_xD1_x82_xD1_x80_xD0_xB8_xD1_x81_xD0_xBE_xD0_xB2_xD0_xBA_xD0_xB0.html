int  windowHeight,
int  frame 
)

Покадрово Рисует анимацию нажатия на кнопку

Аргументы
windowWidthРазмер окна по x
windowHeightРазмер окн по y
frameКадр анимации в дипозоне от 0 до 40
for(int i = 0; i<41; i++){
drawAnimateClick(800, 600, i);
}
Предупреждения
Разрешение экрана может быть любым, функция сама расчитает анимацию.
Данная функция сама очищает экран для следующего кадра!
Возвращает
Выводит на экран анимацию нажатия на кнопку в виде летящего самолёта
